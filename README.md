# OGLDemos
OpenGL Practice


A Repo to share my practice using OpenGL.


Dependencies:
* vcpkg
* vs2020
* imgui (included)
* glfw
* glew


Installation:

1. Download and install vs2020
2. Install vcpkg
   * Install glew using vcpkg
   * Install glfw using vcpkg

Running:

Open visual studio solution and run
